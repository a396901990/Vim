指令：
---------------
    
    set o -vi           # 将shell设置为vi模式  
    set fenc=gbk        # 设置编码格式  
    set nu              # 标记行号
    set nonu            # 取消行号
    set ic              # 搜索时忽略大小写
    set noic            # 搜索时区分大小写

编辑：
----------------
    
插入：

    a       # 光标后插入
    i       # 光标前插入
    o       # 光标的下一行
    O       # 光标的上一行

前撤销/后撤销：

    u           # 向前撤销（回退）
    ctrl+y      # 向后撤销（前进）

复制/剪贴/粘贴：
    
    yy      # 复制行
    nyy     # 复制n行

    dd      # 删除行（剪贴）
    ndd     # 删除n行

    p       # 粘贴到光标之下
    P       # 粘贴到光标之上
